# React Router + Vla Example

This example demonstrates how to use Vla with React Router 7.

## Running the example

```bash
pnpm install
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- Type-safe data layer with Vla
- Server-side rendering with React Router 7
- Route loaders and actions for data fetching and mutations
- Session management with cookies

## Endpoints

- `GET /` - HTML page with posts listing and creation form
- `GET /api/posts` - Get posts as JSON

## Project Structure

- `app/routes/` - React Router route files
- `app/data/` - Data layer with Vla classes
